# Bran (God Mode) CLI

This package provides a command-line interface to install and manage the Bran (God Mode) persona prompts for various AI coding agents.

## Installation

```bash
pip install .
```

## Usage

```bash
# Install the base persona globally
bran install

# Install a specific persona
bran install --persona trader

# Install to a local project instead of globally
bran install --scope local

# Update to the latest personas from the repository
bran update
```
